# 📁 智能文件分类整理器

一键将杂乱文件夹内的文件按扩展名自动归类，支持自定义规则、预览模式、拖拽操作。

## ✨ 功能特点
- 自动分类图片、文档、压缩包等常见文件类型
- 支持用户自定义分类规则（保存为 JSON）
- 预览模式：先看结果再执行，安全可靠
- 拖拽文件夹到窗口，操作便捷
- 实时日志，清晰记录每个文件的去向
- 防止文件名冲突（自动添加序号）


## 🛠️ 技术栈
- Python 3.8+
- tkinter（GUI）
- tkinterdnd2（拖拽支持）
- os, shutil, json, threading（标准库）

## 🚀 快速开始
1. 安装依赖：`pip install tkinterdnd2`（可选）
2. 运行：`python file_organizer.py`
3. 选择文件夹 → 预览 → 开始整理
